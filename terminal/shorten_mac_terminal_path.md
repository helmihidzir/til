# Shorten Mac Terminal Path

`export PS1='\W$ '` = `til$ `

`export PS1="\u@\h \w> "` = `username@host /full/path/to/directory>`

`export PS1="\u@\h \W> "` = `username@host directory>`
`

For permanent change, edit at `~/.bash_profile` and put one of the command above.
